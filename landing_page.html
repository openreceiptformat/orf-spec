A Privacy-First Standard for
Digital Receipts

250 billion paper receipts are printed annually. They're toxic, easily lost, and current digital solutions violate privacy. ORF is the open alternative.

Try the Validator Read the Spec View on GitHub

The Problem

  • Toxic Waste: Thermal receipts contain BPA and cannot be recycled
  • Privacy Violations: Current digital receipt apps require accounts and cloud storage
  • No Interoperability: Each merchant uses proprietary formats that don't work together
  • Costly for Merchants: Paper and printer maintenance costs $500-2,000/year per location

The Solution

  • Privacy-First Design: Receipts stored locally on user's device
  • Interoperability: Any app can read any ORF receipt
  • Merchant-Friendly: Easy to generate from existing POS systems
  • Future-Proof: Extensible schema supports new features without breaking changes
